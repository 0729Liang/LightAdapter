

组件化 - 架构层

> `component_basic` 底层支持，无业务耦合，提供统一依赖管理、基础服务建设、组件化+MVVM基础架构实现

- Common 公共类库依赖
- Resource 业务无关公共资源打包
- MVVM 基础架构 by MVVMLight
- 页面路由管理 by ARouter
- 服务暴露和自动发现 by ARouter
- 图片加载服务 by Glide
- JSON 解析服务 by Gson
- 网络请求 by RxJava2 + Retrofit + OkHttp

[ARouter](https://github.com/alibaba/ARouter)
